# templates

個人開発で使うテンプレート集です。要件定義書などのドキュメント、設定用の YAML ファイル、GitHub の PR テンプレートなど、開発のたびに使い回すひな形をまとめています。

## テンプレート一覧

### ドキュメント

| テンプレート                   | 概要                                                                                                               |
| ------------------------------ | ------------------------------------------------------------------------------------------------------------------ |
| [requirements-spec.md](docs/requirements-spec.md) | システム開発の要件定義書テンプレート。プロジェクト概要から機能・非機能要件、スコープ、リスク管理、承認までを網羅。 |
| [readme-template.md](docs/readme-template.md) | ポートフォリオ向け README テンプレート。概要・デモ・技術スタック・セットアップ・工夫した点などをまとめた構成。 |

### GitHub

| テンプレート                                                          | 概要                                                                                                 |
| --------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------- |
| [pull_request_template.md](.github/pull_request_template.md)           | Pull Request 用テンプレート。概要・変更内容・動作確認・チェックリストなどをまとめたレビュー向けひな形。 |

### CI

| テンプレート                                            | 概要                                                                                             |
| ------------------------------------------------------- | ------------------------------------------------------------------------------------------------ |
| [github-actions-ci.yml](ci/github-actions-ci.yml)       | GitHub Actions 用の汎用 CI ワークフロー。lint → test の構成で、言語に合わせて書き換えて使うひな形。 |
