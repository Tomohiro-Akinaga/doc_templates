# templates

個人開発で使うテンプレート集です。要件定義書などのドキュメント、GitHub の PR テンプレート、GitHub Actions の CI 設定など、開発のたびに使い回すひな形をまとめています。

## テンプレート一覧

### ドキュメント

| テンプレート                   | 概要                                                                                                               |
| ------------------------------ | ------------------------------------------------------------------------------------------------------------------ |
| [requirements-template.md](docs/requirements-template.md) | システム開発の要件定義書テンプレート。プロジェクト概要から機能・非機能要件、スコープ、リスク管理、承認までを網羅。 |
| [readme-template.md](docs/readme-template.md) | README テンプレート。概要・デモ・技術スタック・セットアップ・工夫した点などをまとめた構成。 |

### GitHub

| テンプレート                                                          | 概要                                                                                                 |
| --------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------- |
| [pull_request_template.md](.github/pull_request_template.md)           | Pull Request 用テンプレート。概要・変更内容・動作確認・チェックリストなどをまとめたレビュー向けひな形。 |

### CI

| テンプレート                                            | 概要                                                                                             |
| ------------------------------------------------------- | ------------------------------------------------------------------------------------------------ |
| [github-actions-ci-template.yml](ci/github-actions-ci-template.yml)       | Node.js (npm) プロジェクト向けの GitHub Actions CI ワークフロー。push / PR をトリガーに lint → test を実行する構成。 |
