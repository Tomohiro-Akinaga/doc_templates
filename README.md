### 要件定義書の使い方

1. `要件定義書.md` をコピーし、プロジェクト名をファイル名に含めて保存する
2. `{{` で全文検索し、プレースホルダをすべて埋める
3. 要件ID（BR / F / NFR / DR / SC）を使い、設計書・テスト計画書とトレースする
4. ステータスを「承認済」にする際は、改訂履歴（表紙下）と承認欄（15章）を更新する
