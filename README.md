# templates

個人開発で使うテンプレート集です。要件定義書などのドキュメント、設定用の YAML ファイル、GitHub の PR テンプレートなど、開発のたびに使い回すひな形をまとめています。

## テンプレート一覧

### ドキュメント

| テンプレート                   | 概要                                                                                                               |
| ------------------------------ | ------------------------------------------------------------------------------------------------------------------ |
| [要件定義書.md](要件定義書.md) | システム開発の要件定義書テンプレート。プロジェクト概要から機能・非機能要件、スコープ、リスク管理、承認までを網羅。 |

### GitHub

| テンプレート                                                          | 概要                                                                                                 |
| --------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------- |
| [pull_request_template.md](.github/pull_request_template.md)           | Pull Request 用テンプレート。概要・変更内容・動作確認・チェックリストなどをまとめたレビュー向けひな形。 |
